# MindMic Native Desktop
The enterprise Linux system daemon mapping completely decoupled IPC bridging endpoints dynamically routing recording loops explicitly over AGS components dynamically decoupled across the screen workspace! 

## How It Works
1. **Background Daemon `daemon.py`**: Executes entirely invisible in standard background processes allocating TCP payloads actively passing states backwards and forwards hooking explicitly locally onto PyAudio chunk strings dynamically connecting over your defined configurations!
2. **AGS Presentation (`ags/`)**: Wayland JS wrappers dynamically mapping structural visuals across `gtk-layer-shell` generating completely fluid draggable floating widgets seamlessly overriding standard UI focus locks passively mapping hardware metrics.

## Installation / Usage
Map the isolated global paths inside the core `.env` configurations.
```bash
cp .env.example .env
# Setup absolute paths binding back up into your VENV instances.
```
```bash
# Launch background loop logic safely decoupled
python cli.py start
# Execute specific floating UI toggles
python cli.py toggle-ui
```
